## Redux
state를 관리하는 전용 장소(store)에서 상태 관리

- 재생 / 정지 기능
- Progress bar 기능
- volume 기능
- 이전 곡 / 다음 곡
- Play Mode (랜덤 플레이, 한 곡 반복, 전체 반복)
- play list 재생시간 구하기
